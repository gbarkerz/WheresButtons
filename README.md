# Where's Buttons?

This app demonstrates how UI elements that are not built using appropriate semantic controls can result in the app being unusable for many people. For example, two of the tappable UI elements showing face emojis below cannot be used by people who use their voice to interact with apps on iOS, nor by people who use their keyboards to interact with apps on Windows.

For details on the rationale behind the app, please visit [Hey Cinderella, where's Buttons?](https://www.linkedin.com/pulse/hey-cinderella-wheres-buttons-guy-barker-3r2de)
